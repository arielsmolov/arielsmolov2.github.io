Final Project

For my final project, I worked on a complex application using the following technologies and features:

  • Xcode: The main IDE used to develop the app.
  • Using SwiftUI and .swift Views: Utilized SwiftUI for creating responsive and interactive UI components.
  • Using Firebase: Integrated Firebase for backend services like authentication and data storage (very challenging to implement).
  • Variables: Made use of various variables for state and data handling.
  • Private Variables: Implemented private variables to encapsulate data within views and controllers.
  • Countless Structs: Utilized numerous Swift structs to define data models and views.
  • Countless Let Statements: Employed many `let` statements for defining constants and immutable data.
  • 3 Views/Pages: Developed three main pages for the app:
    • Login Page: Allows users to log in with credentials.
    • Sign In Page: Provides an interface for new users to create accounts.
    • Notes View: A functional page for taking and displaying notes (still under development).
